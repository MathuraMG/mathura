A compilation of a select UX prototypes and wireframes.

This was made to use on a 16.5"x10.5" screen in flight. It is assumed that the screen is at an arms length from the user. The aim is to be able to order a platter of items with minimal effort while being able to personalise the dishes at the same time.

While designing this, I took into consideration that not all people who travel in flight know what the food items are, so when you click any item, it will show you a picture as well as let you order more than 1 if needed.

Count up/Down Phone App

This app is meant to be used as a count up/down app where the user should be able to count down to an event or count up to a limit. I tried to provide an inspiration to the same by revealing a picture that conveys the purpose of counting.
